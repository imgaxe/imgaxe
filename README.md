## Install

[npm][]:

```sh
npm install imgaxe
```

[yarn][]:

```sh
yarn add imgaxe
```


## Usage

```js
const Imgaxe = require('imgaxe');

const imgaxe = new Imgaxe();

console.log(imgaxe.renderName());
// script
```
